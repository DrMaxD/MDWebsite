MDWebsite is a website designed to advertise myself (Max Debin) and demonstrate by web development abilities. This project was inspired by multiple friends who have all created and recomended creating a personal website. This project was created with HTML, CSS There are two versions of the website. A static website and a dynamic one using flask conected to a postgres database. The four main pages are an about me page, projects and experiences, my resume, as well as a contact me page where when the user fills out the information, an email will be sent to me containing the contents of the message.
